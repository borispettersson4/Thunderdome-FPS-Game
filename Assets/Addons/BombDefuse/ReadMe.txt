Bomb Defuse Game Mode addons for MFPS 2.0 
Version 1.1.9

Require MFPS 2.0 v1.4++

Integration:

- After import the package in your MFPS 2.0 project go to (Top Bar) MFPS -> Addons -> BombDefuse -> Enable (if is disable)
- Then in each map scene (not menu scenes) do this: (Top Bar) MFPS -> Addons -> BombDefuse -> Integrate.
- You will see a new prefab with some cubes that represent the bombs, with this you are ready to go and reskin and improve the system.
